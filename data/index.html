MIDI Controller

MIDI command elements are be separated by a space, multiple (max 32) commands can be sent by separating them with a comma.
Format: COMMAND CHANNEL BYTE1 BYTE2 (BYTE2 is optional). Both bytes can be VAR.
Example: CC 1 127, PC 1 127 0
Available commands:
CC - Control Change
PC - Program Change
NOTE_ON - Note On
NOTE_OFF - Note Off
KEY_PRESSURE - Key Pressure
PITCH_BEND - Pitch Bend
CHANNEL_PRESSURE - Channel Pressure
VAR_INC - Increment VAR, Channel is ignored, BYTE1 is the amount to increment by, BYTE2 is ignored.
VAR_DEC - Decrement VAR, Channel is ignored, BYTE1 is the amount to decrement by, BYTE2 is ignored.
